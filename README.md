# voids

`voids` is a scientific Python package for pore network modeling (PNM), focused on
reproducibility, PoreSpy interoperability, and validation-first development.

## v0.1 scope

- Canonical network data model
- PoreSpy-style dictionary import
- Static petrophysics (absolute/effective porosity, connectivity metrics)
- Single-phase incompressible flow solve
- Directional absolute permeability estimation
- HDF5 serialization + provenance metadata

See `spec_v0_1.md` for the technical specification.
